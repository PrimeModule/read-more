Setup

Minimum use of javascript and flat structure make it easy to setup.

Basic usage:

JavaScript

$(.js-read-more-content).readMore();

HTML

<div class="js-read-more">
  <a href="#" class="js-read-more-trigger">Read more</a>
  <div class="js-read-more-content" hidden>
    ...content...
  </div>
</div>

Advanced usage:

HTML - Trigger as anchor with content id in href

<div>
  <a href="#example_id_1" class="js-read-more-trigger">Read more</a>
</div>
...
<div>
  <div id="example_id_1" class="js-read-more-content">
    ...content...
  </div>
</div>

HTML - Trigger as button with content id in data-rm-link attribute

<div>
  <button data-rm-link="#example_id_1" class="js-read-more-trigger">Read more</button>
</div>
...
<div>
  <div id="example_id_1" class="js-read-more-content">
    ...content...
  </div>
</div>

HTML - Trigger with specified action: toggle(default), open, close in data-rm-action attribute.

<div>
  <button data-rm-link="#example_id_1" data-rm-action="open" class="js-read-more-trigger">Open</button>
</div>
...
<div>
  <div id="example_id_1" class="js-read-more-content">
    ...content...
    <button data-rm-link="#example_id_1" data-rm-action="close" class="js-read-more-trigger">Close</button>
  </div>
</div>
